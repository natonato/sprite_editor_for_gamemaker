# React 프로젝트용 스켈레톤

## 이것저것

-   router를 통한 페이지 이동
-   header/body/footer 분리
-   left/right sidebar 분리
-   버튼을 통한 modal 호출/제거
-   Top 버튼

## gh pages action으로 자동화

-   main branch에 push/merge시 자동으로 실행
-   레포가 private일 때 github pro가 아니면 요금 발생 가능

## gh pages SPA 적용

-   [spa-github-pages](https://github.com/rafgraph/spa-github-pages)참고
-   404 page에서 연결해주는 방식
